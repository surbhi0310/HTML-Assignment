1. HTML

    2. Primary Use Cases Key Features/Benefits
    Structuring web content (text, image, etc.)
    • Standard markup language for creating web pages.
    • Provides basic document structure.
    • Supports multimedia elements (images, videos, forms, etc.).
    • SEO-friendly markup.
    • Easy to learn and use.

  2. CSS

    3. Primary Use Cases Key Features/Benefits
    Styling and layout of Web pages.
    • Controls layout, fonts, colors, and spacing.
    • Separates content from design.
    • Responsive design capabilities with media queries.
    • Allows animation and transition effects.
    • Supports frameworks like Bootstrap for faster development.

  3. Javascript

    4. Primary Use Cases Key Features/Benefits
    Client-side web interactivity and logic
    • Adds dynamic content to websites (e.g., form validation, interactive elements).
    • Supports both front-end and back-end (Node.js) development
    • Versatile: runs in browsers and on servers.
    • Integrates well with APIs for asynchronous data loading (AJAX, Fetch API).
    • Broad ecosystem with libraries and frameworks (React, Angular, Vue).
