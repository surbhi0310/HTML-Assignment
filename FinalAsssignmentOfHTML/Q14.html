HTML Tags

  1. Document Structure Tags:
    1. DOCTYPE - Defines the document type (HTML5).
    2. html - The root element of an HTML document.
    3. head - Contains meta-information about the document (e.g., title, links, etc.).
    4. body - Contains the visible content of the webpage.

  2. Metadata Tags:
    1. Title - Defines the title of the document (shown in the browser tab).
    2. meta - Specifies metadata like character set, viewport settings, and keywords.

  3. Text Formatting Tags:
    1. h1 to h6 - Heading tags (from largest to smallest).
    2. p - Defines a paragraph.
    3. br - Inserts a line break (self-closing tag).
    4. hr - Inserts a horizontal line (self-closing tag).
    5. b - Bold text.
    6. strong - Important (bold) text.
    7. i - Italic text.
    8. em - Emphasized (italic) text.
    9. mark - Highlighted text.
    10. small - Smaller text.
    11. del - Strikethrough (deleted) text.
    12. ins - Inserted(underlined)text.
    13. sub - Subscript text
    14. sup - Superscript text.
    15. blockquote - Block of quoted text.
    16. q - Inline quote.
    17. pre - Preformatted text (preserves spaces and line breaks).
    18. abbr -Abbreviation.
    19. time - Defines a time or date.

  4. Links and Media:
    1. a - Anchor tag (for hyperlinks).
    2. img - Embeds an image (self-closing tag).
    3. video - Embeds a video.
    4. audio - Embeds audio content.
    5. source - Defines media source for video or audio.
    6. iframe - Embeds another webpage within the current page.

  5. List Tags:
    1. ul - Creates an unordered list(bulleted).
    2. ol - Creates an ordered list(numbered).
    3. li - Defines a list item inside ul or ol tag.
    4. dl - Definition list.
    5. dt - Definition term.
    6. dd - Definition description.

  6. Table Tags:
    1. table - Creates a table.
    2. tr - Defines a table row.
    3. th - Defines a table header cell
    4. td - Defines a table data cell.
    5. caption - Table caption.

  7. Forms:
    1. form - Creates a form for user input.
    2. input - Creates an input field(self-closing tag).
    3. textarea - Multi-line text input field.
    4. button - Clickable button
    5. select - Drop-down list
    6. option - Option in a drop-down list
    7. label - Label for an input element
    8. fieldset - Grups related form element.
    9. legend - Title for the fieldset.
